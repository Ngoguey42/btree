# Usage

Clone [bentov](https://github.com/gbelouze/bentov) and [oracle](https://github.com/gbelouze/oracle) and link it

```bash
# from btree/
mkdir vendors
cd ..
git clone https://github.com/gbelouze/bentov
git clone https://github.com/gbelouze/oracle
cd btree/
cd vendors
ln -s ../../bentov
ln -s ../../oracle
cd ..
```
